# vue-manager-spa

- 本管理系统模板是基于vue-element-admin加以一些样式和功能的修饰来完成的.

## 技术栈以及组件库

- vue-cli: 快速构建项目

- vuex：状态管理

- vue-router: 路由管理，实现spa单页面开发

- axios: 异步请求数据

- sass: css预处理器

- element-ui: ui组件库

- js-cookie：处理cookie的API

npm install
