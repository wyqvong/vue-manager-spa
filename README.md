# vue-manager-spa

## 技术栈以及组件库

- vue-cli: 快速构建项目

- vuex：状态管理

- vue-router: 路由管理，实现spa单页面开发

- axios: 异步请求数据

- sass: css预处理器

- element-ui

## Project setup
```
npm install
```
